Города

Основание — ванильным Колоколом, команда /city create. Граница квадратная, растёт с числом жителей. Приглашения, топы по числу жителей и активности. Территория без приватов — мир остаётся открытым.

Чат

Обычное сообщение — локально, слышат игроки поблизости. С восклицательным знаком — глобально, слышит весь сервер. С решёткой — только жители своего города.

Экономика города

За каждого нового жителя город получает очки. Очками мэр покупает баффы для жителей внутри границы: регенерацию, скорость добычи, скорость передвижения, сытость, стойкость к урону.

Специализация

Мэр выбирает один раз, навсегда: шахтёрская, фермерская, лесная или рыболовная. Город пассивно копит соответствующий ресурс, мэр периодически забирает его в окне города.

Дикие земли

Дальше 5000 блоков от центра мира монстры сильнее — не запрет, а предупреждение зайти туда осознанно.

Прочее

Кровати работают только на территории своего города. Голосовой чат Plasmo Voice. Карта сервера в браузере — BlueMap.
